# 🚀 FACE-ATTENDANCE-RECOGNITION

## 📌 Overview

This project is an **AI-powered Face Recognition Attendance System** that automates attendance marking using facial recognition technology. It detects faces in real-time and records attendance efficiently.

---

## ✨ Features

* 📸 Single Image Face Recognition
* 📂 Bulk Image Processing
* 🧠 Face Embedding using DeepFace (FaceNet)
* 📊 Interactive Dashboard using Streamlit
* ✅ Automatic Attendance Marking

---

## 🛠️ Technologies Used

* Python
* Streamlit
* DeepFace
* TensorFlow / Keras
* Pandas

## ▶️ How to Run Locally

```bash
git clone https://github.com/misy-git/FACE-ATTENDANCE-RECOGNITION
cd FACE-ATTENDANCE-RECOGNITION
pip install -r requirements.txt
streamlit run dashboard.py
```

---

## ⚠️ Note

* This project requires a **webcam** for face detection.
* It must be run on a **local machine**.
* Deployment on platforms like Netlify is not supported for this type of application.
